# AI Resume Analyzer

## Overview

AI Resume Analyzer is a web-based application developed using Flask and Python that helps job seekers evaluate their resumes. Users can upload a resume in PDF format and select a target job role. The system analyzes the resume, compares the skills present in the resume with the required skills for the selected role, and generates a resume score along with feedback.

---

## Features

* Upload Resume in PDF Format
* Multiple Job Role Selection
* Automatic Skill Extraction from Resume
* Skill Matching Analysis
* Resume Score Calculation
* Missing Skill Identification
* Resume Improvement Feedback
* Responsive and User-Friendly Interface

## How It Works

1. Upload a resume in PDF format.
2. Select a desired job role.
3. The system extracts text from the uploaded resume.
4. Skills are compared with the predefined skills required for the selected role.
5. A resume score is generated based on matching skills.
6. Missing skills and improvement suggestions are displayed.
